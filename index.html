HTML Reference

Head Elements

Tag Name Description
<head> head Defines information about the document
<title> title Defines the title of a document
<base> base Defines a default address or a default target for all links on a page
<link> link Defines the relationship between a document and an external resource
<meta> metadata Defines metadata about an HTML document
<script> script Defines a client-side script
<style> style Defines style information for a document

Head Attributes
Attribute Name Description
profile URL Used to specify the URL of the meta-data. Deprecated in HTML5.
media media_query Used to specify the device/media, the media resource is optimized for.
type text/css Used to specify the type of file within the <style>element.
